你现在需要帮我开发一套「海外私密办公加密聊天SaaS系统」，严格按照以下需求开发，不允许自由发挥、不允许增加社交功能。
项目架构：前后端分离
前端部署在 Vercel，后端部署在 Google Cloud Ubuntu 服务器。
项目用途：面向海外用户的付费私有团队聊天室，用于Stripe HK收款审核与正式商用，全站英文，屏蔽中国大陆访问。
一、项目定位
纯办公私密团队聊天工具，无陌生人匹配、无公开社交、无娱乐功能。用户付费订阅后可以创建私有聊天室、邀请成员团队沟通。
二、前端需求（Vercel）
1. 全站英文UI，简洁专业SaaS风格
2. 包含页面：首页、定价页、隐私政策页、联系页、注册登录、会员中心、聊天界面
3. 页面必须标注：18+ Only、No Stranger Matching、Private Office Chat Only
4. 所有接口请求统一请求后端HTTPS域名
5. 禁止存放任何密钥、订单敏感数据
三、后端需求（GCP Ubuntu）
1. 用户邮箱注册、登录、权限管理
2. 会员订阅体系：月付/年付，自动识别会员状态与到期时间
3. Stripe订阅支付完整对接：创建订阅、支付成功回调、开通权限、续费、取消订阅
4. WebSocket实时私有聊天室功能：创建房间、邀请加入、实时消息、仅私有团队可见
5. 完整数据库结构：用户表、订阅表、房间表、消息表
6. 后端开启CORS，仅允许前端域名访问
7. 全程HTTPS，环境变量存放所有密钥
四、强制禁止功能
严禁开发：陌生人匹配、用户推荐、社交广场、动态、关注、交友、娱乐功能。一经发现全部删除。
五、输出要求
请你分模块输出：前端完整代码、后端完整代码、部署步骤、环境配置教程、Stripe对接教程。代码必须可直接部署运行，无BUG、无冗余功能。
